# Computing for a Sustainable Future

## Project Overview
This is a complete static website developed for a Conestoga College English Technical Communication group assignment. The project explores the intersection of **Computer Science and Climate Change**, highlighting how computational technologies can be used to monitor, model, and mitigate environmental crises.

The website is designed with a focus on professional academic communication, clear organization, and modern aesthetic principles.

## How to View
This website is built using only standard **HTML5** and **CSS3**. To view the website:
1.  Navigate to the project folder.
2.  Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, or Edge).
3.  The website is fully usable offline as all styles and content are local.

## Website Structure
*   `index.html`: Home page featuring a project overview and goal statement.
*   `data.html`: Academic data, climate statistics, and sector-specific emission analysis.
*   `solutions.html`: Deep dive into CS solutions such as AI, Smart Grids, and Precision Agriculture.
*   `ethics.html`: Detailed section on Technical Communication ethics and Creative Commons licensing.
*   `team.html`: Team member profiles and contribution matrix.
*   `references.html`: Categorized academic references formatted in APA style.
*   `styles/styles.css`: Centralized design system and responsive layout rules.
